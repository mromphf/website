## Tooling

Built with `Vue.js` using `npm` for installing deps and building code, etc.

### Install Dependencies
```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
